# Bitam Telecom — تطبيق المتجر

تطبيق ويب يعرض للزبائن السلع والأسعار، يسمح لهم بالطلب، وينقص المخزون تلقائياً (أو يخفي السلعة عند نفاذها). قابل للتثبيت على الهاتف كتطبيق حقيقي (PWA)، وقابل للتحويل إلى ملف APK.

---

## 1) التجهيز على حاسوبك

1. ثبّت **Node.js** (نسخة 18 فما فوق) من https://nodejs.org
2. افتح مجلد المشروع بالـ Terminal ونفّذ:
   ```
   npm install
   ```

---

## 2) إعداد قاعدة البيانات (Firebase) — مجاني

هذا يجعل المنتجات والطلبات محفوظة بشكل دائم ومشتركة بين كل من يفتح الرابط.

1. اذهب إلى https://console.firebase.google.com وسجّل الدخول بحساب Google
2. **Add project** → اختر اسم (مثلاً `bitam-telecom`) → أكمل الخطوات (يمكنك تعطيل Google Analytics)
3. من القائمة الجانبية: **Build → Firestore Database** → **Create database** → اختر "Start in production mode" → اختر أقرب موقع (مثلاً `eur3`)
4. اذهب إلى **⚙️ Project settings** (بجانب "Project Overview") → انزل لأسفل إلى **Your apps** → اضغط أيقونة **</>** (Web)
5. سمّه (مثلاً "bitam-web") → **Register app**
6. سينسخ لك Firebase كود فيه `firebaseConfig = {...}` — انسخ هذه القيم والصقها في الملف:
   ```
   src/firebase.js
   ```
   بدّل كل قيمة (apiKey, authDomain, projectId...) بالقيمة الحقيقية من Firebase.
7. من نفس القائمة الجانبية: **Firestore Database → Rules** → الصق محتوى ملف `firestore.rules` الموجود في هذا المشروع → **Publish**

---

## 3) التجربة محلياً

```
npm run dev
```
افتح الرابط الذي يظهر (عادة `http://localhost:5173`) في المتصفح. أول مرة، التطبيق يملأ المنتجات الافتراضية تلقائياً في Firestore.

**الرمز السري الافتراضي للوحة التحكم: `1234`** — غيّره فوراً من: زر ⚙️ أعلى الصفحة → أدخل الرمز → تبويب "الإعدادات".

---

## 4) نشر الموقع على الإنترنت (رابط دائم) — مجاني عبر Vercel

1. أنشئ حساب على https://github.com (إذا ما عندكش) وحمّل المشروع كـ repository جديد:
   - أنشئ repo فارغ على GitHub
   - في مجلد المشروع:
     ```
     git init
     git add .
     git commit -m "بداية المشروع"
     git branch -M main
     git remote add origin رابط_الـ_repo_تاعك
     git push -u origin main
     ```
2. اذهب إلى https://vercel.com → سجّل الدخول بحساب GitHub
3. **Add New → Project** → اختر الـ repo تاعك → اضغط **Deploy**
4. بعد دقيقة تقريباً، يعطيك Vercel رابط دائم مثل:
   `https://bitam-telecom.vercel.app`

هذا الرابط هو الذي تعطيه للزبائن.

---

## 5) تحويله إلى تطبيق يُثبَّت على الهاتف (PWA)

المشروع مجهز مسبقاً بـ `vite-plugin-pwa`، فبمجرد نشره على Vercel:

- **زبون Android (Chrome)**: يفتح الرابط → تظهر رسالة "إضافة إلى الشاشة الرئيسية" أو من قائمة ⋮ → "Install app" / "تثبيت التطبيق"
- **زبون iPhone (Safari)**: يفتح الرابط → زر المشاركة (Share) → "Add to Home Screen"

بعدها تظهر أيقونة التطبيق في هاتفه وتفتح بدون شريط المتصفح، مثل أي تطبيق عادي.

---

## 6) تحويله إلى ملف APK حقيقي (اختياري)

إذا تحب تعطي الزبائن ملف APK يثبتونه مباشرة (أو تنشره في Google Play):

1. اذهب إلى https://www.pwabuilder.com
2. الصق رابط موقعك (من Vercel) → **Start**
3. بعد الفحص، اضغط **Package for Stores** → اختر **Android**
4. حمّل ملف الـ APK الجاهز وأرسله للزبائن، أو اتبع تعليمات الموقع لرفعه على Google Play

---

## ⚠️ ملاحظة أمان مهمة

قواعد Firestore في هذا المشروع (`firestore.rules`) مفتوحة مؤقتاً لتسهيل الإعداد الأول. **قبل ما تعطي الرابط لعدد كبير من الزبائن**، من الأفضل تقييدها أكثر (مثلاً منع أي شخص من قراءة طلبات الزبائن الآخرين). إذا حبيت، اطلب مني لاحقاً "شدد قواعد الأمان" وأنا نجهزلك نسخة أدق.

---

## هيكل المشروع

```
src/
  App.jsx        ← كل واجهة المتجر ولوحة التحكم
  firebase.js     ← إعدادات الاتصال بقاعدة البيانات (عدّلها بمعلوماتك)
  index.css       ← الخطوط والتنسيق العام
public/
  icons/icon.svg  ← أيقونة التطبيق
firestore.rules   ← قواعد الأمان لقاعدة البيانات
```
